# Cisco Packet Tracer :

## Introduction
Cisco Packet Tracer is a network simulation tool used to design, configure, and troubleshoot virtual networks without physical hardware.

##  Purpose
- Learn networking concepts  
- Practice configurations  
- Simulate real-world networks  

##  Key Features
- Real-time mode (normal operation)
- Simulation mode (step-by-step packet flow)
- CLI & GUI configuration
- Supports basic protocols (TCP/IP, DHCP, HTTP)

##  Components
- **End Devices:** PC, Laptop, Server  
- **Network Devices:** Router, Switch  
- **Connections:** Ethernet, Fiber  
- **Workspace:** Area to build network  

##  Basic Workflow
1. Add devices  
2. Connect them  
3. Assign IP addresses  
4. Test using `ping`  

##  Simulation Mode
- Visualizes packet movement  
- Helps in debugging and learning  

##  Common Tasks
- LAN setup  
- Router configuration  
- VLAN basics  
- Troubleshooting  

##  Advantages
- No hardware required  
- Easy to use  
- Great for beginners  

## Limitations
- Limited real-world accuracy  
- Not all protocols supported

# How End devices communicate

PC1 -> switches -> router -> fireWall -> ModeM -> (LAN -> WAN) ie Internet -> ModeM -> FireWall -> router -> switches -> PC2
